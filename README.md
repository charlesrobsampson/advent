# advent

stuff I wrote for [advent of code](https://adventofcode.com/)

# usage

simply run the day you want to see

you can run it from root<br />
`node 2022/1.js`

or the year<br />
`node 1.js`

if you want to specify part a or b, you can<br />
`node 1.js b`

if you don't specify it'll run both
